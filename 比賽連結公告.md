# 匹克球比賽 · 連結公告文字

給選手、觀眾看的連結說明。**直接複製下面框框裡的文字**貼到 LINE 群組、海報或簡報即可。

## 版本 B：每個連結一句話（貼海報／簡報）

```
● 觀眾看板 → https://pickball.onrender.com
  現場大螢幕、觀眾用。即時比分、各級積分榜、淘汰對戰表、賽果，全部自動更新。

● 選手報名 → https://pickball.onrender.com/register
  選手自己填姓名＋分級，送出就進報名名單，不用排隊登記。

● 計分操作 → https://pickball.onrender.com/control
  主辦方專用（不要外發）。抽籤分組、逐球計分，結果即時同步到觀眾看板。
```
