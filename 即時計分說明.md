# 匹克球即時計分系統 · 使用說明

一個線上匹克球比賽系統，分成**三種畫面**：選手自己報名、主辦方計分、觀眾看即時比分。所有畫面**即時同步**——計分台按一下，觀眾看板和其他手機同一秒跟著更新。

---

## 一、正式網址

系統已部署在雲端，只有一個網址，**後面加不同結尾**就是不同畫面：

| 畫面 | 網址 | 給誰 |
|------|------|------|
| 📊 觀眾看板 | https://pickball.onrender.com | 觀眾、所有人 |
| ✍️ 選手報名 | https://pickball.onrender.com/register | 參賽選手 |
| 🎮 計分操作 | https://pickball.onrender.com/control | 主辦方（你） |

> 💡 手機、電腦、平板都能開，開瀏覽器輸入網址即可，不用安裝任何 App。

## 二、完整流程（賽前 → 比賽中 → 賽後）

```
選手報名 (/register)
      │  填姓名＋分級，即時進入名單
      ▼
主辦抽籤 (/control → 多組賽事分配)
      │  按「帶入報名」把名單抓進來 → 抽籤配對 → 產生賽程
      ▼
計分 (/control)
      │  每球點一下、或直接輸入整局比分
      ▼
觀眾看板 (/) 即時顯示比分、積分榜、對戰表、賽果
```

### 步驟 1 — 選手報名（`/register`）

1. 選手用手機開 `https://pickball.onrender.com/register`
2. 輸入**姓名**，選**分級**（2.0 級 / 3.0 級）
3. 按「送出報名 →」
4. 名字立刻出現在下方「目前報名名單」，所有裝置同步看到
5. 填錯了 → 點名字上的 **×** 移除

### 步驟 2 — 抽籤分組（`/control`）

1. 主辦方開 `https://pickball.onrender.com/control`
2. 在首頁底部點 **「🏆 多組賽事分配」**
3. 設定賽事名稱、每局分數（11／15）、計分規則
4. 在「2.0 級名單」「3.0 級名單」卡片右上角，點 **「帶入報名（N）」**
   → 剛剛報名的選手自動填入格子（也可手動修改）
5. 按 **「🎲 抽籤分組 →」**
   → 每級 10 人隨機配成 **5 對固定搭檔**，5 對互相循環賽

### 步驟 3 — 計分（`/control`）

抽籤後會看到賽程表（2 場地 × 5 輪）。找到要打的那場，按 **「開始計分」** 進入計分板。

計分板有**兩種模式**（右下角切換）：

- **點擊**：每贏一球，就點「贏得那球的隊伍」一下，系統自動處理發球、換發、換邊、唱分
- **輸入**：直接輸入該局最終比分（例如 11:7），適合事後補登

其他按鈕：
- **回上一步**：記錯了可以復原
- **交換顯示**：左右兩隊對調顯示位置

一場打完 → 按「返回賽程」回到賽程表，打下一場。全部循環賽打完 → 按 **「產生淘汰賽」**（前 4 對晉級，準決賽 → 決賽）。

### 步驟 4 — 觀眾看板（`/`）

觀眾開 `https://pickball.onrender.com`，上方有四個分頁：

| 分頁 | 內容 |
|------|------|
| **即時比分** | 正在進行那場的即時比分、發球方、局數（有比賽時自動跳到這頁） |
| **積分榜** | 各級 5 對循環賽排名（勝場優先、淨分次之） |
| **對戰表** | 抽籤配對、賽程、淘汰賽對戰圖與冠軍 |
| **賽果** | 所有已完成比賽的比分紀錄 |

觀眾看板**全唯讀**，沒有任何操作按鈕，不怕觀眾誤按。

## 三、計分規則

| 項目 | 說明 |
|------|------|
| **賽制** | 單打 / 雙打 |
| **每局分數** | 11 分 或 15 分，**需領先 2 分** 才算贏該局 |
| **局數** | 三局兩勝（Best of 3） |
| **換邊** | 每局開始交換場邊（畫面會提示「請交換場邊」） |

### 兩種計分規則

- **發球得分制（Side-out，傳統制）**
  只有**發球方**得分；接發方贏球不得分。
  雙打有第 1／第 2 發球員，唱分為「發球方分數－接發方分數－第幾發球員」。

- **Rally 制（每球得分）**
  每一球都得分，輸球方換發球權；一樣要領先 2 分才贏。

## 四、比賽當天小提醒

- ⏰ **先喚醒系統**：免費雲端主機閒置 15 分鐘會休眠，下次開網址要等約 50 秒。比賽前你先自己開一次 `/control` 把它叫醒。
- 📱 **一機一畫面**：計分的人開 `/control`，投影／大螢幕開 `/`（觀眾看板），選手手機開 `/register`。互不干擾又即時同步。
- 🔄 **報名名單會在重新部署時清空**：正常使用（比賽當天不動程式）不受影響。
- 🖥 **不用開自己電腦**：系統在雲端 24 小時運作，你電腦關機也沒關係。

## 五、常見問題

**Q：報名了，計分台看不到名字？**
A：報名同步是自動的，但名單要**手動按「帶入報名」**才會填進抽籤格子。確認你在 `/control` →「多組賽事分配」畫面，看名單卡片右上角的按鈕。

**Q：只有一個網址，怎麼分三個畫面？**
A：同一個網址，後面加結尾：`/` 觀眾、`/register` 報名、`/control` 計分。

**Q：計分點錯了怎麼辦？**
A：按計分板的「回上一步」，可連續復原。

**Q：手機連不上 / 轉圈圈很久？**
A：主機剛從休眠喚醒，等約 50 秒重新整理即可。
